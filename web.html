Desenvolvimento Web

O desenvolvimento web é o processo de criar e manter sites e aplicações web. Este campo abrange uma variedade de tarefas e tecnologias que visam garantir que os sites sejam funcionais, acessíveis e esteticamente agradáveis.

Durante as aulas de desenvolvimento web, aprendemos como é o processo completo de criação de um site, desde o design da interface até a implementação do código. A área de desenvolvimento web possui três especializações principais: Front-end, Back-end e Full-stack.

O que aprendemos?

Durante as aulas de desenvolvimento web, exploramos o processo completo de criação de um site. Isso inclui desde o design da interface até a implementação do código.

Plataforma usada

No início, aprendemos a usar o Figma para criar um protótipo do design do site. Para desenvolver o site com código, usamos o Visual Studio Code, onde aprendemos os seguintes fundamentos:

HTML:

HTML (HyperText Markup Language) estrutura o conteúdo no site. Funciona como o esqueleto de um corpo humano, criando a estrutura dos textos e títulos.

  • A sigla HTML significa HyperText Markup Language.
  • É identificado pelo navegador através da extensão .html.
  • Possui uma estrutura padrão com tags específicas para definir elementos.
Elemento Descrição
<html> Define o documento como HTML
<head> Contém metadados e links para estilos
<body> Contém o conteúdo visível da página

CSS:

CSS (Cascading Style Sheets) é responsável pelo estilo visual do site, tornando-o mais bonito e agradável ao usuário. Ele funciona como a pele e os cabelos de um corpo, dando forma e cor ao site.

  • CSS significa Cascading Style Sheets.
  • Aplica estilos a elementos HTML, como cores, fontes e layouts.
  • Permite criar um design responsivo, adaptando o layout para diferentes dispositivos.
Propriedade Descrição
color Define a cor do texto
font-size Define o tamanho da fonte
margin Define o espaço ao redor dos elementos

JavaScript:

JavaScript é uma linguagem de programação que dá vida ao site, permitindo a funcionalidade e interatividade. Foi criada pela Netscape para aumentar a interação do usuário com as páginas web.

  • JavaScript é a linguagem de programação mais popular no desenvolvimento web.
  • Responsável por praticamente qualquer tipo de dinamismo nas páginas.
  • Permite criar aplicações web complexas, como Gmail, Google Maps e Google Docs.
  • Utiliza frameworks e bibliotecas como React, Angular e Vue.js para simplificar o desenvolvimento.
Elemento Descrição
function Define uma função em JavaScript
var Declara uma variável
document.getElementById() Seleciona um elemento HTML por ID

Pilha e Fila:

De forma rápida, pilha é o conceito de que o último item a ser adicionado é o primeiro a ser retirado. Fila é o conceito de que a primeira pessoa a chegar é a primeira a ser atendida.

  • Pilha: LIFO (Last In, First Out). Exemplo: uma pilha de pratos, onde o último prato colocado é o primeiro a ser retirado.
  • Fila: FIFO (First In, First Out). Exemplo: uma fila de pessoas em um banco, onde a primeira pessoa a chegar é a primeira a ser atendida.
